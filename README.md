Vestigo
=======

File manager in Vala and GTK3

How to install?
````
git clone https://github.com/simargl/vestigo
cd vestigo
make install
gtk-update-icon-cache /usr/share/icons/hicolor
glib-compile-schemas /usr/share/glib-2.0/schemas
````
Dependencies:
````
gtk+-3.0 >= 3.10
gee-0.8
poppler-glib
````
